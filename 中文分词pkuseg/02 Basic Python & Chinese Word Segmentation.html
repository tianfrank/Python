Python基础

Python 简介

Python是一种广泛使用的解释型、高级编程、通用型编程语言，由Guido van Rossum创造，第一版发布于1991年。Python的设计哲学强调代码的可读性和简洁的语法（尤其是使用空格缩进划分代码块，而非使用大括号或者关键词）。相比于C++或Java，Python让开发者能够用更少的代码表达想法。不管是小型还是大型程序，该语言都试图让程序的结构清晰明了。

与其它动态类型编程语言一样，Python拥有动态类型系统和垃圾回收功能，能够自动管理内存使用，并且支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。其本身拥有一个巨大而广泛的标准库。

Python 解释器本身几乎可以在所有的操作系统中运行。Python的其中一个解释器CPython是用C语言编写的、是一个由社群驱动的自由软件，当前由Python软件基金会管理。

-- Wikipedia

Python有着活跃的社区，提供了海量的程序包，越来越获得数据科学家和机器学习研究者的青睐。

Python语法基础

Python程序有两种执行方式：

  • 交互式：一边编写一边执行 (REPL, Read-Eval-Print Loop)
  • 脚本式：编写好完整的程序并存储于.py文件中，然后执行

Hello, World!

打开交互式Python终端，在命令行中输入

python

你的第一个Hello World程序

xxxxxxxxxx
print("Hello, World!")

print将给定的对象打印出来，而双引号表示对象是一个字符串(string)。print可以接受任意多个对象，对象间键默认以空格分开

xxxxxxxxxx
print("Hello,", 'World!')

注意Python中的字符串有多种表示方法，单引号、双引号与3个单/双引号均表示字符串。

Python标识符

Python中的标识符用于表示对象、函数、模块、类或者其它对象，一些保留的关键词用于支撑以上功能的表示。

xxxxxxxxxx
a                 # 对象
def a(): pass     # 定义函数
import a          # 导入模块
class a: pass     # 定义类

Python语句

Python默认每行是一个语句，代码块通过缩进来组织，:表示开启一个新的缩进代码块，使用;可以显式结束一个语句，并在一行内输入多个语句。

xxxxxxxxxx
person = ['Alice', 'Ben']
for x in person:
    if x == 'Alice':
        print(x)
    else:
        print('not Alice')

Python中以#开始的行表示注释，是对程序的解释，后面的内容不会被执行。

Python安装

Anaconda

Python具有很好的平台兼容性，在几乎所有的操作系统上可以安装（官方网站, Windows Store, 各Linux发行版的包管理器都有提供）。同时，Python的PyPI (Python Package Index)提供Python程序包的分发。但很多用到C/C++扩展的Python程序包，安装时需要额外的编译过程，如果包管理者没有提供预编译好的二进制包的话，在Windows平台上很容易出问题。

Python有众多的发行版，针对不同的应用场景进行了优化。我们这里会以Anaconda发行版进行演示。Anaconda可以看成一个“应用商店”，除了Python，还提供很多其它的程序比如Perl，R等。同时这个“应用商店”还是一个“插件商店”，有很多基于Python的程序包。

从这里下载Miniconda进行安装，Miniconda是Anaconda的精简版本，随附了较少的程序包，但可以后续再安装。Windows和MacOS安装过程和常规程序安装差不多。

由于conda的官方软件源速度较慢，后续安装程序包会花费很多时间，建议按照这里配置镜像源；对于PyPI也是如此，参照这里配置。

虚拟环境

版本管理在Python开发中是很重要的。Python和Python的程序包处于持续的开发过程中，很多API并没有后向兼容性的保证，而且不同版本运行可能会获得不同的结果。这对于科学计算来说不是很理想的。Python自身提供了虚拟环境的管理方案，每个虚拟环境都是独立的，可以保证版本。Anaconda也提供了这样的机制。

打开Anaconda Prompt或者终端，输入以下命令创建一个名为tutorial的虚拟环境，并安装python3.6和numpy。Anaconda提供的numpy是基于Intel MKL优化过的，计算效率更高。

xxxxxxxxxx
conda create -n tutorial python=3.6 numpy

使用conda activate和conda deactivate激活或者停止虚拟环境，环境的名称会在终端中prompt前显示。

xxxxxxxxxx
conda activate tutorial
conda deactivate

要查看可用的虚拟环境，可以使用

xxxxxxxxxx
conda env list

安装程序包时，使用conda install命令安装conda提供的程序包，使用pip install安装PyPI提供的程序包。一般建议当conda没有提供对应的程序包时，才使用pip进行安装。不同虚拟环境中pip也是隔离的。避免使用两个工具安装同样的包。

xxxxxxxxxx
conda install jupyter
# pip install jupyter

要查看虚拟环境中可用的程序包，可以使用

xxxxxxxxxx
conda list # 查看虚拟环境中的所有包，区分pip安装的包
# pip list # 查看虚拟环境中的所有python程序包，不与conda安装的包区分显示

中文分词工具

Jupyter Notebook

交互性在数据分析中是很重要的，而Python默认的交互环境并不是非常友好，我们这里会使用jupyter notebook来。jupyter notebook是基于web的Python交互式环境，具有很多适合数据分析的特性，比如在线修改、数据展示、输出结果存储等。

在先前的例子中，我们已经安装了jupyter。激活对应环境后，要开启jupyter notebook，在对应终端中输入

xxxxxxxxxx
jupyter notebook

之后jupyter notebook server会在终端前台运行，并自动打开网页，当前终端的工作目录为jupyter的根目录。

停止jupter可以通过网页中的Quit按钮，也可以在终端中输入CTRL+C。

jieba与pkuseg

我们这里会对比jieba和pkuseg两个分词工具。jieba默认模式是基于词典的匹配方案，同时也支持基于隐马尔可夫的模型。pkuseg使用基于CRF的分词方案。首先使用pip安装两个工具包

xxxxxxxxxx
pip install jieba pkuseg

然后在jupyter中新建一个notebook，开始之后的对比。
